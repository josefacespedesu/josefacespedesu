## 🧠 Caso de Estudio: Predicción de Gasto en Clientes E-commerce

### 📌 Descripción
Desarrollo de un modelo de Machine Learning supervisado para predecir el monto de compra de clientes en un entorno e-commerce, utilizando variables demográficas y de comportamiento digital.

### ⚠️ Desafío principal
El principal desafío fue lograr un modelo con buen desempeño sin caer en sobreajuste, considerando la necesidad de preprocesar adecuadamente los datos y seleccionar el algoritmo más adecuado entre varias alternativas.

### 💡 Solución propuesta
Se implementó un pipeline completo de Machine Learning que incluyó limpieza de datos, codificación de variables, escalamiento, entrenamiento de múltiples modelos de regresión y optimización mediante validación cruzada y ajuste de hiperparámetros.

### 🛠️ Herramientas utilizadas
Python, Pandas, Scikit-learn, Statsmodels, GridSearchCV

### 📚 Aprendizajes
- Importancia del preprocesamiento en el rendimiento del modelo  
- Diferencias entre modelos y su impacto en la generalización  
- Uso de regularización para evitar sobreajuste  
- Interpretación de métricas como MAE, RMSE y R²  

### 📊 Impacto / Resultados
- Desarrollo de un modelo predictivo funcional  
- Mejora del rendimiento mediante optimización  
- Selección de un modelo final (Lasso) con buen equilibrio entre precisión y generalización  

### 🧠 Habilidades aplicadas
- Análisis de datos  
- Modelamiento predictivo  
- Evaluación de modelos  
- Pensamiento analítico  
- Resolución de problemas  

### 🎯 ¿Por qué este proyecto?
Este proyecto fue seleccionado porque representa de manera integral mi capacidad para desarrollar soluciones de Machine Learning, desde el preprocesamiento hasta la evaluación y selección de modelos, aplicando criterios técnicos y de negocio.
